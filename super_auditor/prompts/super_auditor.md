# Prompt Super Auditor
Use com entradas dos relatórios do MCP Dev Flow.

## Descrição
Prompts para auditoria automática de arquitetura, testes, segurança e custos.

## Exemplo de Prompt
- Validar SOLID: "Verificar se classes seguem SRP, OCP, LSP, ISP, DIP."
- Cobertura TDD: "Garantir >80% cobertura de testes unitários."
- Segurança: "Auditar compliance LGPD e vulnerabilidades OWASP."
- Custos: "Calcular ROI para refatoração de módulo legado."
- Arquitetura: "Validar bounded contexts em DDD."

Use com entradas dos relatórios do MCP Dev Flow.
