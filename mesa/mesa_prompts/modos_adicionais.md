# Modos adicionais
- Modo Auditor
- Modo Construtor

## Modo Auditor
- Descrição: Foca em validação e auditoria de qualidade.
- Uso: Aplicar após rodadas para checar conformidade com padrões.
- Output: Relatório de auditoria.

## Modo Construtor
- Descrição: Gera código e artefatos baseados no consenso.
- Uso: Após consenso, construir skeleton e guias.
- Output: Código inicial e documentação.
- Exemplo: "Gere classe User em Java com JPA annotations."
