# Exemplo Task 01 – Setup Env

## Título
Configurar Ambiente de Desenvolvimento Java 21 com Spring Boot

## Descrição
Esta tarefa configura o ambiente necessário para desenvolvimento de aplicações Java usando Spring Boot 3.2.5, JDK 21 e ferramentas relacionadas. Inclui instalação de dependências, configuração de IDE e validação.

## Prompts
- Instalar JDK 21 (OpenJDK ou Oracle JDK).
- Configurar variáveis de ambiente JAVA_HOME e PATH.
- Instalar Maven 3.9+ ou Gradle.
- Configurar IntelliJ IDEA com plugins para Spring Boot.
- Criar um projeto Spring Initializr básico e executar para validar.

## Saídas Esperadas
- Ambiente funcional: Comando `java -version` retorna JDK 21.
- Projeto de teste: Aplicação Spring Boot rodando em localhost:8080.
- Logs de instalação e configuração.

## Critérios de Aceitação
- JDK instalado e configurado corretamente.
- Projeto Spring Boot criado e executado sem erros.
- Documentação de passos realizada.
