# Exemplo Task 02 – Casos de Uso

## Título
Criar Casos de Uso para Módulo de Regressão Econômica

## Descrição
Esta tarefa define e documenta os casos de uso para o módulo de regressão econômica no sistema Poupador Calcula Economia. Inclui identificação de atores, fluxos principais e alternativos, e validação com stakeholders.

## Prompts
- Identificar atores: Usuário, Sistema, Administrador.
- Definir casos de uso: Executar Regressão, Visualizar Resultados, Configurar Parâmetros.
- Descrever fluxos: Pré-condições, passos, pós-condições.
- Criar diagramas UML de casos de uso.
- Validar com equipe de negócio.

## Saídas Esperadas
- Documento de casos de uso em Markdown.
- Diagramas em formato PNG ou SVG.
- Lista de requisitos funcionais derivados.

## Critérios de Aceitação
- Todos os atores e casos de uso identificados.
- Fluxos completos e validados.
- Aprovação da equipe de negócio.
