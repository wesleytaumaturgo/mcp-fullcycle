## Descrição
Na Fase 1 (Intake), coletamos requisitos iniciais e contexto do projeto para estabelecer a base para o desenvolvimento.

## Prompts para Agente
- Colete informações básicas: Nome do sistema, stack tecnológica, objetivos principais.
- Identifique stakeholders e necessidades iniciais.
- Valide viabilidade básica (ex.: compatibilidade com infraestrutura existente).

## Saídas Esperadas
- Documento de intake preenchido.
- Lista de requisitos preliminares.

## Critérios de Aceitação
- Contexto claro e aprovado pelos stakeholders.

## Exemplo de Output
- Documento de intake: "Sistema: Poupador Calcula Economia, Stack: Java/Spring, Objetivos: Otimizar cálculos."
